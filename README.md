# Ecommerce-Web Project

***
### Introduction

It's an attempt to emulate parts of an ecommerce enterprise UX with maximum functionalities. Current build missing connections to payment gateway feature.


### Technology Used :

- Angular 4
- Hibernate
- Spring Boot
- Java
- Maven
- Postgres (Database)
